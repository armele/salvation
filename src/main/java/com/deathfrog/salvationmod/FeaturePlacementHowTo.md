# Feature Definition
Define a feature at: src\main\resources\data\salvation\worldgen\configured_feature
* For example: blightwood_tree.json or blightwood_tree_tall.json

# Feature Placement
Provide definitions for how this feature will get placed in the world at: src\main\resources\data\salvation\worldgen\placed_feature\blightwood_grove_dense.json
* For example: blightwood_grove_sparse.json or blightwood_grove_dense.json

# Biome Configuration
The placement detail can be referenced from the biome definiton files at: src\main\resources\data\salvation\worldgen\biome
* For example: corrupted_swamp.json
